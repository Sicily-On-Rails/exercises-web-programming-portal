Esempio di dichiarazione ed assegnazione di variabili in javascript

  • Dichiarazione di una variabile con valore di un numero intero


    var score; //Dichiarazione della variabile attracerso la parola chiave "var"
    score = 100; //Assegnazione di un valore alla variabile
    console.log(score); //Metodo che ritorna il valore della variabile
    Visualizza Esempio

  • Dichiarazione di una variabile con valore di una stringa


    var message; //Dichiarazione della variabile con nome "message" attraverso la parola chiave "var"
    message ="Ciao mondo!!!"; //Assegnamio alla varibile "message" del testo racchiuso tra doppi apici
    console.log(message); //Metodo che ritorna il valore della variabile
    Visualizza Esempio
